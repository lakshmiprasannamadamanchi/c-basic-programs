# c-basic-programs

Description

This repository contains basic C programming problems that help in understanding fundamental concepts such as conditional statements, loops, and number-based logic. These programs are useful for beginners to build strong programming and problem-solving skills.

Programs Included

- Even or Odd Number
- Positive, Negative, or Zero
- Sum of N Natural Numbers
- Factorial of a Number
- Fibonacci Series
- Prime Number Check
- Reverse a Number
- Palindrome Number
- Armstrong Number
- Sum of Digits

Purpose

The purpose of this repository is to practice C programming fundamentals and strengthen logic building before solving advanced coding problems.
